The Good Websites logo — globe-and-smiley mark + Excon wordmark. The mark is embedded as a data URI, so it's fully portable.

```jsx
<Logo size="lg" />                     {/* mark + ink word, light bg */}
<Logo colorway="light" />              {/* cream globe, for dark sections */}
<Logo colorway="onSun" />              {/* on a gold block */}
<Logo markOnly size="sm" />            {/* just the globe-and-smiley */}
```

Colourways: `ink`, `light`, `onSun`. Sizes: `sm`, `md`, `lg`. Set `markOnly` for the mark alone (favicon, avatars, loaders).
