Labelled text input with hint and error states. Pops a hard offset shadow on focus.

```jsx
<Input label="Your name" placeholder="Posy Field" required />
<Input label="Email" type="email" error="That doesn't look right" />
```

Props: `label`, `hint`, `error`, `required`, plus all native input attributes (`type`, `value`, `onChange`, `placeholder`, `disabled`…).
